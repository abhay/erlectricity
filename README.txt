= Erlectricity - An Erlang -> Ruby bridge that operates using erlang ports

== DESCRIPTION:

Erlectricity is a library for ruby that implements a an API with which to interact with Erlang through the ports system in a more natural way. It allows you to integrate ruby code as you would any other process in the erlang ecosystem.

== REQUIREMENTS:

* concurrent (0.0.2) gem

INSTALL:

    sudo gem install erlectricity
